## Настройка после установки

1. Night Mode: `on`
1. Settings:
   - Chat Settings:
     - Themes: `Tinted`
     - Messages: `Send with Ctrl+Enter`
   - Advanced:
     - Ask download path for each file: `on`
     - Automatic media download:
       - Files: `off`
     - Spell checker / Manage dictionaries:
       - English: `on`
       - Русский: `on`
